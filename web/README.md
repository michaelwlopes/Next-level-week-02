## Give Classes
- Register a class with the following items:
  - Teacher's info
    - name
    - avatar (URL, future work: upload photo inside application)
    - contact (WhatsApp number)
    - bio
  - Classes Info
    - Subject (Select among Brazilian high-school subjects)
    - class cost
    - Schedule
      - Week day
      - time (from - to)
